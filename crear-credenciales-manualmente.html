Crear Base de Datos manualmente

Necesitamos saber de antemano:

  • nombreBaseDatos : El nombre que le queremos poner a la base de datos.
  • nombreHost : Donde nos queremos conectar. Puede ser localhost, ejemplo.com, etc.
  • nombreUsuarioBaseDatos : El nombre de un usuario con privilegios para poder crear y modificar la base de datos del chat.
  • contraseñaUsuarioBaseDatos : Contraseña del usuario.

Una vez tenemos toda la información a mano, podemos proceder modificando los nombres en rojo por nuestra información:

  1. Vamos a nuestro administrador de bases de datos y Creamos la BBDD:

    "CREATE DATABASE nombreBaseDatos"

  2. Le añadimos la tabla para guardar el historial de mensajes:

    CREATE TABLE nombreBaseDatos.`ciberchat` ( `numeroMensaje` INT(11) NOT NULL AUTO_INCREMENT , `usuario` VARCHAR(30) NOT NULL DEFAULT 'Desconocid@' , `mensaje` VARCHAR(300) NOT NULL , `hora` VARCHAR(30) NOT NULL , PRIMARY KEY (`numeroMensaje`)) ENGINE = InnoDB;

  3. Le añadimos la tabla de usuarios conectados:

    CREATE TABLE nombreBaseDatos.`usuariosConectados` ( `usuario` VARCHAR(30) NOT NULL ) ENGINE = InnoDB;

  4. Por último, creamos en la carpeta php un archivo llamado "cred.php" y le añadimos la siguiente información:

    <?php $host = 'nombreHost'; $nombreUsuario = 'nombreUsuarioBaseDatos'; $contrasenhaUsuario = 'contraseñaUsuarioBaseDatos'; $nombreBBDD = 'nombreBaseDatos'; ?>
